# KPI Definition Report

# Loan Approval Analysis Project

---

# Overview

This document defines the Key Performance Indicators (KPIs) used to evaluate the bank's loan approval process. These KPIs provide measurable insights into application volume, approval performance, applicant financial profiles, credit risk, and customer segmentation.

The KPIs defined in this document will serve as the foundation for the Power BI dashboard and executive reporting developed in later phases of the project.

---

# KPI Objectives

The primary objectives of these KPIs are to:

* Monitor loan application performance.
* Measure loan approval efficiency.
* Evaluate applicant financial characteristics.
* Assess credit risk.
* Support management decision-making.
* Enable business performance monitoring through interactive dashboards.

---

# KPI Categories

The KPIs are organized into five business categories:

1. Loan Portfolio KPIs
2. Financial KPIs
3. Credit Risk KPIs
4. Asset KPIs
5. Customer Segmentation KPIs

---

# 1. Loan Portfolio KPIs

These KPIs provide an overall summary of the bank's loan application process.

| KPI                     | Definition                                 | Business Purpose                             |
| ----------------------- | ------------------------------------------ | -------------------------------------------- |
| Total Loan Applications | Total number of loan applications received | Measures business volume and customer demand |
| Approved Loans          | Number of approved loan applications       | Tracks successful loan approvals             |
| Rejected Loans          | Number of rejected loan applications       | Monitors rejection trends                    |
| Loan Approval Rate (%)  | Approved Loans ÷ Total Applications × 100  | Measures lending efficiency                  |
| Loan Rejection Rate (%) | Rejected Loans ÷ Total Applications × 100  | Evaluates rejection patterns                 |

---

# 2. Financial KPIs

These KPIs describe the financial characteristics of loan applicants.

| KPI                         | Definition                          | Business Purpose                          |
| --------------------------- | ----------------------------------- | ----------------------------------------- |
| Average Annual Income       | Average annual income of applicants | Understand customer income levels         |
| Average Loan Amount         | Average requested loan amount       | Measure customer borrowing requirements   |
| Total Loan Amount Requested | Sum of all requested loan amounts   | Estimate lending demand                   |
| Average Loan Term           | Average repayment period            | Understand customer repayment preferences |
| Maximum Loan Amount         | Highest requested loan amount       | Identify high-value loan requests         |

---

# 3. Credit Risk KPIs

These KPIs evaluate the creditworthiness of applicants.

| KPI                              | Definition                                      | Business Purpose                                |
| -------------------------------- | ----------------------------------------------- | ----------------------------------------------- |
| Average CIBIL Score              | Average applicant CIBIL score                   | Measure overall credit quality                  |
| Highest CIBIL Score              | Maximum CIBIL score                             | Identify strongest credit applicants            |
| Lowest CIBIL Score               | Minimum CIBIL score                             | Identify highest-risk applicants                |
| Approval Rate by Credit Category | Approval percentage within each credit category | Evaluate lending decisions across risk segments |

---

# 4. Asset KPIs

These KPIs summarize the asset profile of applicants.

| KPI                             | Definition                             | Business Purpose                    |
| ------------------------------- | -------------------------------------- | ----------------------------------- |
| Average Residential Asset Value | Average residential property value     | Assess applicant property ownership |
| Average Commercial Asset Value  | Average commercial asset value         | Measure business-related assets     |
| Average Luxury Asset Value      | Average luxury asset value             | Evaluate wealth indicators          |
| Average Bank Asset Value        | Average value of bank-held assets      | Assess financial stability          |
| Total Asset Value               | Combined value of all applicant assets | Measure overall applicant wealth    |

---

# 5. Customer Segmentation KPIs

These KPIs compare approval performance across different applicant groups.

| KPI                                   | Definition                                   | Business Purpose                                        |
| ------------------------------------- | -------------------------------------------- | ------------------------------------------------------- |
| Approval Rate by Education            | Approval percentage by education level       | Evaluate approval trends across educational backgrounds |
| Approval Rate by Self-Employment      | Approval percentage by employment type       | Compare salaried and self-employed applicants           |
| Approval Rate by Number of Dependents | Approval percentage by dependent category    | Understand the impact of family size                    |
| Approval Rate by Loan Term            | Approval percentage across repayment periods | Evaluate lending preferences by loan duration           |
| Approval Rate by Income Category      | Approval percentage across income groups     | Assess lending performance by income segment            |
