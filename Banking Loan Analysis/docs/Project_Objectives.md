# Project Objectives

## Project Goal

The goal of this project is to analyze historical loan application data to understand the factors influencing loan approval decisions and support business stakeholders in improving lending strategies through data-driven insights.

---

# Business Objectives

The project aims to achieve the following business objectives:

1. Understand the current loan approval process.
2. Identify the major factors affecting loan approval.
3. Improve approval consistency across applicants.
4. Reduce financial risk associated with loan approvals.
5. Support business decision-making using analytical insights.
6. Increase operational efficiency within the loan approval process.

---

# Analytical Objectives

To achieve the business objectives, the analysis will:

* Examine applicant demographics.
* Analyze applicant income distribution.
* Study loan amount trends.
* Evaluate the impact of credit history.
* Analyze approval trends across education levels.
* Analyze approval trends by employment status.
* Compare approval rates across property areas.
* Calculate business KPIs.
* Create an interactive Power BI dashboard.

---

# Success Criteria

The project will be considered successful if it:

* Identifies key factors influencing loan approval.
* Produces meaningful business insights.
* Answers important stakeholder questions.
* Provides actionable recommendations.
* Includes SQL analysis and Power BI dashboards.
* Is documented professionally for GitHub and portfolio presentation.

---

# Deliverables

The project will deliver:

* Cleaned dataset
* Data cleaning documentation
* Exploratory Data Analysis report
* SQL business analysis scripts
* KPI calculations
* Interactive Power BI dashboard
* Business insights report
* Executive summary
* GitHub-ready project documentation
